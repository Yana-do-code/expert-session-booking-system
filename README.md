# Real-Time Expert Session Booking System

A full-stack web application for booking sessions with experts in real time. The backend is built with Node.js, Express, and MongoDB (Mongoose). The frontend is a React single-page application that communicates with the API to browse experts, view available time slots, and book sessions.

## Project Structure

- `server/` - Node.js + Express + Mongoose backend
- `client/` - React frontend

## Features (planned)

- Browse and search expert profiles
- View available session slots per expert
- Real-time booking with conflict prevention
- View and manage personal bookings
